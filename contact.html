FAQ

Pertanyaan yang
sering ditanyakan

Q
Apakah EduTrack sudah bisa digunakan langsung?
EduTrack saat ini masih dalam tahap pengembangan sebagai project kuliah. Kami membuka komunikasi untuk mendapatkan masukan, kolaborasi, dan rencana implementasi ke depannya.
Q
Apakah EduTrack gratis untuk universitas?
Sebagai project akademik, EduTrack bersifat open-source. Diskusi lebih lanjut mengenai implementasi dan dukungan teknis dapat dilakukan melalui formulir kontak di atas.
Q
Teknologi apa yang digunakan dalam EduTrack?
EduTrack dibangun menggunakan HTML/CSS/JavaScript di frontend, Node.js di backend, MySQL sebagai database, dengan REST API dan Chart.js untuk visualisasi data akademik.
Q
Bisakah saya bergabung sebagai kontributor?
Tentu! Kami menyambut siapapun yang ingin berkontribusi — baik sebagai developer, designer, maupun researcher. Pilih topik "Bergabung sebagai Kontributor" di formulir atas.
Q
Berapa lama proses respons dari tim EduTrack?
Tim kami biasanya merespons dalam 24 jam kerja. Untuk urusan mendesak, Anda dapat menghubungi langsung melalui WhatsApp di nomor yang tertera.
Q
Apakah data mahasiswa aman di sistem EduTrack?
Keamanan data adalah prioritas utama. EduTrack menggunakan sistem autentikasi berbasis peran, enkripsi data, dan tidak pernah membagikan data kepada pihak ketiga tanpa izin.
